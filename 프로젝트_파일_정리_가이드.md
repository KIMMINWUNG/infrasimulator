## 📋 파일별 설명

### ✅ **필수 파일 (절대 삭제 금지)**

#### 개발 필수
- `src/` - 소스 코드
- `public/` - 정적 파일
- `package.json` - 프로젝트 설정
- `index.html` - 메인 HTML
- `vite.config.js` - 빌드 설정
- `tailwind.config.js` - 스타일 설정

#### 오프라인 배포 필수
- `server.cjs` - 로컬 웹 서버
- `create-offline-package.cjs` - 배포 패키지 생성
- `시뮬레이터_실행.bat` - 사용자 실행 파일
- `public/xlsx.full.min.js` - XLSX 라이브러리 (로컬)

---

### ⭐ **중요 파일 (수정 시 주의)**

#### 오프라인 핵심
```javascript
// public/calculation.worker.js
self.importScripts('/xlsx.full.min.js');  // CDN 아님!

// src/main.jsx
import '@fortawesome/fontawesome-free/css/all.min.css';  // CDN 아님!

// index.html
// Font Awesome CDN 링크 없어야 함!
```
