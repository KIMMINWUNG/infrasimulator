## 📌 개요

이 시뮬레이터는 **완전 오프라인** 환경에서 작동합니다.
- ✅ 인터넷 연결 불필요
- ✅ 모든 리소스(아이콘, 폰트 등) 로컬에 포함
- ✅ 더블클릭으로 간편 실행
- ✅ 다른 컴퓨터로 복사만 하면 바로 사용 가능

---

## 🔧 사전 준비물

### 1. Node.js 설치 파일
**사용자 컴퓨터에 Node.js가 설치되어 있어야 합니다.**

#### 📥 Node.js 다운로드 (개발자용)
1. 인터넷이 되는 PC에서 다운로드:
   - 공식 사이트: https://nodejs.org/
   - **LTS 버전** 다운로드 (예: v20.x.x)
   - Windows용 설치 파일: `node-v20.x.x-x64.msi`

2. USB에 저장:
   ```
   USB/
   ├── node-v20.x.x-x64.msi  ← Node.js 설치 파일
   └── 시뮬레이터/             ← 아래 참조
   ```

---

## 📦 배포 방법

### 방법 1: USB 드라이브로 배포 (추천)

#### Step 1: 필요한 파일들만 복사
다음 파일/폴더만 USB에 복사하면 됩니다:

```
USB/시뮬레이터/
├── dist/                      ← 빌드된 앱 (필수)
├── server.cjs                 ← 로컬 서버 (필수)
├── 시뮬레이터_실행.bat         ← 실행 파일 (필수)
├── package.json               ← 프로젝트 정보 (선택)
├── README.md                  ← 설명서 (선택)
└── 오프라인_배포_가이드.md     ← 이 파일 (선택)
```

**⚠️ 중요: 다음은 복사하지 않아도 됩니다**
- ❌ `node_modules/` (용량 큼, 불필요)
- ❌ `src/` (소스코드, 불필요)
- ❌ `.git/` (버전 관리, 불필요)

#### Step 2: 사용자 PC에 복사
1. USB를 사용자 PC에 연결
2. `시뮬레이터` 폴더를 원하는 위치에 복사
   - 예: `C:\Users\사용자명\Desktop\시뮬레이터`
   - 예: `D:\프로그램\시뮬레이터`

---

### 방법 2: 네트워크 공유 폴더 (내부망)

1. 공유 폴더에 위 파일들 업로드
2. 사용자들이 각자 PC로 복사
3. 실행

---

## 🎮 사용 방법

### 처음 사용 시

#### 1단계: Node.js 설치 (한 번만)
```
💡 Node.js가 이미 설치되어 있다면 건너뛰세요!
```

1. `node-v20.x.x-x64.msi` 더블클릭
2. 설치 마법사 따라하기 (기본 옵션 그대로)
3. 설치 완료 후 **컴퓨터 재시작**

#### 2단계: 시뮬레이터 실행
1. `시뮬레이터_실행.bat` **더블클릭**
2. 자동으로 브라우저가 열리고 시뮬레이터 실행됨
3. 사용 완료 후 명령창에서 `Ctrl+C` 또는 창 닫기

---

### 매번 사용 시

1. `시뮬레이터_실행.bat` **더블클릭**
2. 끝! 😊

## ❓ 문제 해결

### 1. "Node.js가 설치되어 있지 않습니다" 오류

**원인:** Node.js 미설치 또는 PATH 설정 오류

**해결:**
1. Node.js 설치 확인:
   - 명령 프롬프트 열기 (Win+R → cmd)
   - `node --version` 입력
   - 버전이 표시되면 설치됨

2. 표시되지 않으면:
   - Node.js 재설치
   - 컴퓨터 재시작

---

### 2. 브라우저가 자동으로 열리지 않음

**해결:**
1. 명령창에 표시된 주소 확인
2. 브라우저를 직접 열고 주소 입력:
   ```
   http://localhost:3000
   ```

---

### 3. "포트 3000이 이미 사용 중" 오류

**원인:** 다른 프로그램이 3000번 포트 사용 중

**해결:**
1. 기존 서버 종료 후 재실행
2. 또는 `server.js` 파일 열기
3. 3번째 줄 `const PORT = 3000;`을 `const PORT = 3001;`로 변경
4. 저장 후 재실행

---

### 4. 아이콘이 표시되지 않음 (□로 보임)

**원인:** Font Awesome 폰트 파일 누락 (거의 발생하지 않음)

**확인:**
```
dist/assets/ 폴더에 다음 파일이 있는지 확인:
- fa-regular-400-XXXXX.woff2
- fa-brands-400-XXXXX.woff2
- fa-solid-900-XXXXX.woff2
```

**해결:**
- `dist/` 폴더 전체를 다시 복사

---

### 5. Excel 파일 업로드가 안됨

**원인:** 파일 경로 또는 형식 문제

**해결:**
1. Excel 파일이 `.xlsx` 형식인지 확인
2. 파일 이름에 특수문자 제거
3. 파일을 열어있지 않은 상태에서 업로드
