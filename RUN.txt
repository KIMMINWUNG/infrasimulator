로컬 실행 (이 PC에서만 사용)
============================
1. npm install
2. npm run dev
3. 브라우저에서 http://localhost:5173 접속

인증 KEY: .env.local 에 VITE_MASTER_KEY 설정 또는 기본값 사용

관리주체 추가/삭제·명칭 변경 (한 곳만 수정)
-----------------------------------------
public/management_list.csv 한 파일만 수정하면 됩니다.
- 단일 실행(관리감독기관·관리주체 선택), 관리자 모드(필터·시군구별 일괄), 일괄 계산 대상 모두 이 CSV 기준으로 동작합니다.
- 형식: 첫 줄 "관리계획 수립기관,관리주체" / 다음 줄부터 "지자체명,관리주체명"
- 수정 후 브라우저 새로고침만 하면 반영. (배포 시 npm run build 후 재배포)
