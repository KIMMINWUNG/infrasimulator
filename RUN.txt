로컬 실행 (이 PC에서만 사용)
============================
1. npm install
2. npm run dev
3. 브라우저에서 http://localhost:5173 접속

인증 KEY: .env.local 에 VITE_MASTER_KEY 설정 또는 기본값 사용

관리주체 추가/삭제
-----------------
public/management_list.csv 만 수정하면 됩니다.
형식: 첫 줄 "관리계획 수립기관,관리주체" / 다음 줄부터 "지자체명,관리주체명"
수정 후 브라우저 새로고침만 하면 반영. (빌드 후 서버 실행 중이면 npm run build 후 재실행)
