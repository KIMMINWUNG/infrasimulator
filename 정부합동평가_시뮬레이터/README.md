# 정부합동평가 시뮬레이터 (지자체 합동평가 시뮬레이터)

오프라인/로컬 환경에서 동작하는 지자체 합동평가 시뮬레이터입니다.

## 개발 실행

```bash
npm install
npm run dev
```

브라우저에서 http://localhost:5173 접속.

## 내부망 배포 패키지 생성

```bash
npm run build
node create-offline-package.cjs
```

`정부합동평가_시뮬레이터` 폴더가 생성됩니다. 이 폴더 전체를 내부망에 복사해 사용하면 됩니다.

- 상세 사용법: 생성된 폴더 안의 **오프라인_배포_가이드.md** 참고.

## 관리주체 수정

- **한 곳만 수정:** `public/management_list.csv` (형식: 첫 줄 `관리계획 수립기관,관리주체` / 이하 `지자체명,관리주체명`)
- 수정 후 브라우저 새로고침 또는 재배포 시 반영.

**버전:** 2.0.0
