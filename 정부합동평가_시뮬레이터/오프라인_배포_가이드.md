# 정부합동평가 시뮬레이터 — 오프라인 배포 가이드

## 1. 빠른 시작

1. **이 폴더 전체**를 내부망에서 사용할 위치에 복사합니다.
2. **시뮬레이터_실행.bat** 파일을 더블클릭하여 실행합니다.
3. 브라우저에서 표시되는 주소(예: http://localhost:5173)로 접속합니다.

**필수:** PC에 Node.js가 설치되어 있어야 합니다. (인터넷 연결은 불필요)

---

## 2. 주요 기능 (현재 반영 버전)

### 2.1 개별 시뮬레이션

- **관리감독기관·관리주체 선택**: 드롭다운에서 지자체와 관리주체를 선택해 단일 대상만 평가합니다.
- **실적 연도 선택**: 25년 실적(제출 기한 2025.2.28) / 26년 실적(제출 기한 2026.2.28) 중 선택합니다.
- **평가 항목 선택**: 실행계획 제출여부 / 최소유지관리기준 / 충당금 조례 제정 중 계산할 항목을 켜거나 끌 수 있습니다. 선택한 항목에 대해서만 파일 업로드가 필수입니다.
- **민자사업자 제외**: 체크 시 민자사업자 실적을 제외하고 점수를 산정합니다.

### 2.2 상세 DB 다운로드 (개별 실행 후)

최소유지관리기준 평가 후 아래 순서로 다운로드할 수 있습니다.

| 구분 | 설명 |
|------|------|
| 민자사업자 제외 DB | 실적 DB에서 해당 지자체·관리주체 + 민자 제외한 행 |
| 관리그룹 포함 | 고시문에 O로 표시된 조합에 해당하는 실적 행 |
| 관리그룹 제외 | 관리그룹에 포함되지 않는 실적 행 |
| 등급확인 (분모) | 관리그룹 포함 중 등급 확인 대상(점수 분모) |
| 목표등급 만족 (분자) | 등급확인 중 목표등급 만족(점수 분자) |
| 목표등급 불만족 | 등급확인 중 목표등급 미만족 |

- 실행계획 항목 선택 시 **실행계획 미제출** DB도 다운로드 가능합니다.

### 2.3 관리자 모드 (일괄 계산)

- **관리감독기관 필터**: 특정 지자체만 선택하거나 전체로 일괄 계산합니다.
- **시군구(관리주체)별 모드**: 켜면 지자체–관리주체별로 점수·상세 데이터를 산정합니다.
- **실적 연도·평가 항목**: 개별 모드와 동일하게 선택할 수 있습니다.
- **상세 DB 다운로드**: 실행계획 미제출, 민자사업자 제외, 관리그룹 포함/제외, 등급확인, 목표등급 만족/불만족을 **지자체(또는 지자체-관리주체)별 시트**로 한 번에 받을 수 있습니다. 버튼 배치는 개별 모드와 동일한 흐름(민자사업자 제외 → 관리그룹 포함·제외 → 등급확인·목표등급)으로 구성되어 있습니다.

### 2.4 관리주체 명칭 관리 (한 곳만 수정)

- **수정 위치**: 이 폴더 안의 `dist/management_list.csv` 한 파일만 수정하면 됩니다.
- **형식**: 첫 줄 `관리계획 수립기관,관리주체` / 다음 줄부터 `지자체명,관리주체명`
- 관리주체 추가·삭제·명칭 변경 시 위 CSV만 수정한 뒤, **브라우저 새로고침**하면 개별 실행·관리자 모드·일괄 계산 모두에 반영됩니다. (배포본을 다시 만들었다면 해당 폴더를 내부망에 덮어쓰면 됩니다.)

## 3. 파일 구성

| 항목 | 설명 |
|------|------|
| dist | 시뮬레이터 웹 파일(빌드 결과). 브라우저가 여기를 서빙합니다. |
| server.cjs | 로컬 서버. bat 파일이 이걸 실행합니다. |
| 시뮬레이터_실행.bat | 더블클릭으로 서버 실행 후 브라우저 열기 |
| package.json | Node 의존 정보 (서버 실행 시 필요) |
| 오프라인_배포_가이드.md | 본 가이드 |
| README_배포용.txt | 빠른 시작 요약(자동 생성) |

---

## 4. 주의사항

- **Node.js**가 설치되어 있어야 합니다.
- 인터넷이 없어도 동작합니다. 데이터는 모두 사용자 PC에서만 처리됩니다.
- 이 폴더를 USB 등으로 복사해 다른 PC에서 사용할 수 있습니다. (해당 PC에도 Node.js 필요)
- 관리주체·실적 연도·평가 항목 등 변경 사항이 반영된 최신 **정부합동평가_시뮬레이터** 폴더로 내부망을 업데이트하면 됩니다.
